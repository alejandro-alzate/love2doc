Input assistance
Input assistance is provided for the printout, sort, and condition field. The condition field requires to use one of following prefixes:
  • p: to fetch property suggestions (e.g. [[p:Has ...)
  • c: to fetch category suggestions
  • con: to fetch concept suggestions
