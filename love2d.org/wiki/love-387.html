love.quit (Italiano)

Disponibile da LÖVE 0.7.0
Questa/o callback non è supportata/o nelle versioni precedenti.

Callback chiamata quando il gioco viene chiuso.

Funzione

Sinossi

r = love.quit( )

Argomenti

Nessuno.

Restituisce

boolean (Italiano) r
Annullare la chiusura. Se è vero, il gioco non viene chiuso.

Esempio

Questo esempio mostra come evitare che il gioco venga chiuso, utilizzando return in love.quit. 

local quit = true
function love.quit()
    if quit then
        print("Il gioco non è pronto alla chiusura!")
        quit = not quit
    else
        print("Grazie per aver giocato! A presto!")
        return quit
    end
    return true
end
