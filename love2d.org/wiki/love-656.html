love.load (Українська)

Ця функція викликається рівно один раз на початку гри.

Функція

Вигляд

love.load( arg, unfilteredArg )

Аргументи

table arg
Аргументи командного рядка, з якими запущена гра.
Доступне починаючи з LÖVE 11.0
table unfilteredArg
Нефільтровані аргументи командиного рядка, з якими запущена гра (див. #Примітки).

Повертає

Нічого.

Примітки

В LÖVE 11.0 з аргументів забирається назва гри та приєднані флаги командного рядка (якщо вони є), коли запускається не приєднана до програми гра. У попередніх версіях всі аргументи передавалися напряму, без фільтрації.

Приклади

Ось як можна встановлювати змінні та завантажувати ресурси під час завантаження, щоб потім вони були доступні у всіх інших функціях (таких як love.draw). 

function love.load()
   hamster = love.graphics.newImage("hamster.png")
   x = 50
   y = 50
end

function love.draw()
   love.graphics.draw(hamster, x, y)
end
