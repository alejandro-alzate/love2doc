love.window.close (Русский)

Available since LÖVE 0.10.0
This function is not supported in earlier versions.

Закрывает окно. Окно может быть открыто заново при помощи love.window.setMode.

Функции и объекты love.graphics вызовут ошибку, если будут использованы при закрытом окне.
