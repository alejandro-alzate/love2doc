love.mousemoved (Українська)

Доступне починаючи з LÖVE 0.9.2
Ця функція недоступна в попередніх версіях.

Функція зворотного виклику, яка спрацьовує під час пересування миші.

Функція

Вигляд

love.mousemoved( x, y, dx, dy, istouch )

Аргументи

number x
Позиція миші по осі X.
number y
Позиція миші по осі Y.
number dx
Зсув по осі X з часу останнього виклику love.mousemoved.
number dy
Зсув по осі Y з часу останнього виклику love.mousemoved.
Доступне починаючи з LÖVE 0.10.0
boolean istouch
true, якщо функція викликана в результаті дотику по сенсорному екрані.

Повертає

Нічого.

Примітки

Якщо для миші увімкнено відносний режим, аргументи dx та dy в функції будуть обновлятися, але обновлення x та y не гарантується.
