love.window.setFullscreen (Français)

Disponible depuis LÖVE 0.9.0
Ce-tte function n'est pas supporté-e par des versions plus anciennes.

Entre ou sort du mode plein écran (fullscreen). L'écran à utiliser lors du passage en plein écran est le même que celui où est actuellement située la fenêtre, dans le cas où plusieurs moniteurs sont connectés.

Fonction

Synopsis

success = love.window.setFullscreen( fullscreen )

Arguments

boolean (Français) fullscreen
Si il faut entrer ou sortir du mode plein écran.

Returns

boolean (Français) success
True (vrai) si la tentative d'entrer en mode plein écran à réussi, false (faux) autrement.

Fonction

Synopsis

success = love.window.setFullscreen( fullscreen, fstype )

Arguments

boolean (Français) fullscreen
Si il faut entrer ou sortir du mode plein écran.
FullscreenType (Français) fstype
Le type de mode plein écran à utiliser.

Retourne

boolean (Français) success
True (vrai) si la tentative d'entrer en mode plein écran à réussi, false (faux) autrement.

Notes

Si la fenêtre est passé en mode plein écran et si la fenêtre ne correspond pas à une taille des modes d'affichage du moniteur (en mode plein écran normal) ou bien l taille de la fenêtre ne correspond pas à la taille du bureau (en mode plein écran 'bureau'), la fenêtre sera redimensionnée de façon appropriée. La fenêtre va retourner de nouveau à ses dimensions originales lorsque le mode plein écran est quitté en à l'aide de cette fonction.

Exemples

Passer la fenêtre en plein écran en mode bureau (desktop).. 

function love.load()
    love.window.setFullscreen(true, "desktop")
end
