Примеры

Заметьте, что love.keypressed даёт прямое нажатие клавиши, тогда как love.textinput даёт текст, который комбинация клавиш производит.

Возьмём символ "#" для примера (на раскладке США): 

function love.keypressed(key)
    print(key)
end
--> shift
--> 3

function love.textinput(text)
    print(text)
end
--> #
