알아 두기

윈도우즈 환경에서 여러 개의 줄을 입력할 때 몇몇 편집기(메모장 등)에서는 CRLF ("\r\n")만을 새 줄로 취급합니다. 

--예제
f = love.filesystem.newFile("note.txt")
f:open("w")
for i = 1, 10 do
    f:write("This is line "..i.."!\r\n")
end
f:close()
