love.draw (Русский)

Функция, вызываемая в каждом кадре, используется для рисования на экране.

Функция

Вид

love.draw( )

Аргументы

Нет.

Возвращает

Ничего.

Примеры

Рисуется изображение, которое было загружено в love.load (расположение love.graphics.newImage в love.draw привело бы к перезагрузке изображения каждый кадр, что вызвало бы проблемы). 

function love.load()
   hamster = love.graphics.newImage("hamster.png")
   x = 50
   y = 50
end
function love.draw()
   love.graphics.draw(hamster, x, y)
end
