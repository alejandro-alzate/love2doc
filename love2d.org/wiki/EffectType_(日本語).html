EffectType (日本語)

LÖVE 11.0 から使用可能
この列挙型は以前のバージョンでは非対応です。

love.audio.setEffect で対応している各種音響効果です。

定数

chorus
コーラスです。わずかな音程のズレと時間変動値を利用して音声の多重複製物を再生します。太い音や厚みのある音にするために使用します。
EffectWaveform waveform
number phase
number rate
number depth
number feedback
number delay
compressor
コンプレッサです。音量の大きい所と静かな所の振幅を近づけ、時間全体の均一振幅を生成することで、音声のダイナミックレンジを削減します。
boolean enable
distortion
ディストーションです。信号の一部を切り取り、圧縮し音声を歪ませ、さらにクリッピングするまで増幅することで音声を変化させます。
number gain
number edge
number lowcut
number center
number bandwidth
echo
エコーです。秒単位で発生する減衰型反響効果です (ディレイとして知られています)。音量を減少させながら一定間隔で音声を反復再生します。
number delay
number tapdelay
number damping
number feedback
number spread
equalizer
イコライザです。４バンドイコライザー (ローバンド、バンドパス、ハイバンド) で音声の周波数成分を調整します。
number lowgain
number lowcut
number lowmidgain
number lowmidfrequency
number lowmidbandwidth
number highmidgain
number highmidfrequency
number highmidbandwidth
number highgain
number highcut
flanger
フランジャーです。音声の二重複製物を再生します。ミリ秒単位により、位相変化、またはディレイ相当のエフェクトを処理することで、音声の段階的位相変化の効果を発生します。
EffectWaveform waveform
number phase
number rate
number depth
number feedback
number delay
reverb
リバーブです。ミリ秒単位指定による減衰型残響効果です。残響によるサラウンド効果を得るために使用します。
number gain
number highgain
number density
number diffusion
number decaytime
number decayhighratio
number earlygain
number earlydelay
number lategain
number latedelay
number roomrolloff
number airabsorption
boolean highlimit
ringmodulator
リングモジュレータです。振幅変調 (AM) の実装です。入力信号へ単純波形を乗算することで、音量変化、または不調和倍音を生成します。
EffectWaveform waveform
number frequency
number highcut
