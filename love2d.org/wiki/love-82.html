love.window (Русский)

Available since LÖVE 0.9.0
This module is not supported in earlier versions.


Предоставляет интерфейс для изменения и получения информации об окне программы.

Функции

love.window.close (Русский) Закрывает окно. 0.10.0
love.window.getDimensions (Русский) Получает ширину и высоту окна. 0.9.0
love.window.getHeight (Русский) Получает ширину окна. 0.9.0
love.window.getIcon (Русский) Получает иконку окна. 0.9.0
love.window.getTitle (Русский) Получает заголовок окна. 0.9.0
love.window.getWidth (Русский) Получает ширину окна. 0.9.0
love.window.isCreated (Русский) Проверяет создано ли окно. 0.9.0
love.window.isMaximized (Русский) Gets whether the Window is currently maximized. 0.10.2
love.window.maximize (Русский) Разворачивает окно, насколько это возможно. 0.10.0
love.window.setIcon (Русский) Устанавлевает иконку окна. 0.9.0
love.window.setTitle (Русский) Устанавливает заголовок окна. 0.9.0
love.window.showMessageBox (Русский) Отображение диалогового окна поверх LOVE окна. 0.9.2
