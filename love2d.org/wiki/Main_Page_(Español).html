Bienvenido

Como sabrás, LÖVE es un framework para la creación de juegos 2D en el lenguaje de programación Lua. LÖVE es totalmente libre y puede ser usado para cualquier propósito, desde amigables proyectos de código abierto como hobby, hasta malvados proyectos comerciales de código cerrado.

Hola Mundo

Este es el código fuente completo para generar un "¡Hola mundo!" en LÖVE. Al ejecutarlo aparecerá una ventana de 800 por 600 que muestra dicho texto en blanco sobre un fondo negro. 

function love.draw()
    love.graphics.print('¡Hola Mundo!', 400, 300)
end
