KeyConstant (한국어)

각각의 키에 대응되는 문자열입니다. 몇몇 키는 시스템이나 키보드에 따라 지원되지 않을 수도 있으니 주의하세요.

기타
numlock Num-lock 키
capslock Caps-lock 키 불 들어왔을 때가 키가 눌린 것으로, 불 꺼지면 뗀 것으로 처리
scrolllock Scroll-lock 키
rshift 오른쪽 시프트
lshift 왼쪽 시프트
rctrl 오른쪽 컨트롤
lctrl 왼쪽 컨트롤
ralt 오른쪽 알트
lalt 왼쪽 알트
rgui 오른쪽 GUI 키 Windows에선 Windows키. Apple에선 Command 키
lgui 왼쪽 GUI 키 Windows에선 Windows키. Apple에선 Command 키
mode 모드 키 일부 키보드에만 있음
조합키와 토글키
pause 포즈 브레이크 키 일부 환경에서 키가 눌리고 있는 상태라도 뗐다는 메시지를 바로 보내짐
escape 이스케이프 키
help 헬프 키 Apple 키보드에만 있다고 함. Windows에서 F1과 동일한 기능
printscreen 프린트스크린 키 Windows 환경에서는 누르고 있어도 키가 즉시 떼어진 것처럼 처리
sysreq 시스템 리퀘스트 키 프린트 스크린 키
menu 메뉴 키 일부 키보드에만 있음
application 애플리케이션 키 Windows에서 메뉴 나오게 하는 키
power 파워 키 일부 키보드에만 있음
currencyunit 통화 기호 백슬래시 키
undo 취소 키 일부 키보드에만 있음
