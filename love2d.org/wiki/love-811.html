love.isVersionCompatible (Français)

Disponible depuis LÖVE 0.10.0
Ce-tte function n'est pas supporté-e par des versions plus anciennes.

Test si la version est compatible avec la version actuelle de LÖVE.

Fonction

Synopsis

compatible = love.isVersionCompatible( version )

Arguments

string (Français) version
La version (Par exemple : "0.10.1").

Retourne

boolean (Français) compatible
SI la version est compatible.

Fonction

Synopsis

compatible = love.isVersionCompatible( major, minor, revision )

Arguments

number (Français) major
La version majeure, par exemple 0 pour la version 0.10.1.
number (Français) minor
La version mineure, par exemple 10 pour la version 0.10.1.
number (Français) revision
La version de révision, par exemple 1 pour la version 0.10.1.

Retourne

boolean (Français) compatible
Si la version est compatible.
