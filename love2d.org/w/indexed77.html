love.joystick.isOpen

Available since LÖVE 0.5.0 and removed in LÖVE 0.9.0
This function is not supported in earlier or later versions.

Checks if the joystick is open.

Function

Synopsis

open = love.joystick.isOpen( joystick )

Arguments

number joystick
The joystick to be checked

Returns

boolean open
True if the joystick is open, false if it is closed.
