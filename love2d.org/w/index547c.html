socket (Русский)

Начиная с 11.0, LÖVE полностью поддерживает не блокируемые TCP соединения в Windows  


Available since LÖVE 0.5.0
This module is not supported in earlier versions.

Модуль luasocket для взаимодействия с TCP/UDP протоколами. Этот модуль встроен в LÖVE, но для его использования вы его должны загрузить так: 

require("socket")

а лучше вот так: 

local socket = require("socket")

Примечание: Начиная с lua 5.2 LÖVE 11.0, нужно использовать последний метод, потому что luasocket больше не записывается в глобальную таблицу после загрузки.

Примечание: При использовании следующих функций: connect/read/write, или socket.sleep происходит приостановка основного потока LÖVE, их использование не очень хорошая идея, используйте не блокирующие функции, если это возможно.
