Post by rude » Sat Feb 13, 2016 3:56 pm

Greetings.

So I thought it would be nice to design new LÖVE icons, as the current icons are severely outdated.

Here they are.
new_icons.png (38.99 KiB) Viewed 26147 times
The main version of the icon (left) is used everywhere, except on OSX which gets extra sex appeal (right), because Apple. (iOS has an icon similar to the OSX one, but with a rounded square mask instead of bordered circle).

I tried to make the OSX version also work on Windows (etc), but it didn't look that good, since the icon typically appears in small sizes, where it's difficult to be recognizable with too much gradient detail going on.

Conversely, I tried to make the Windows (etc) icon work on OSX, but that looked a little bit too simple, and therefore "foreign" to OSX.

Hence there are two different, but related, sets of icons.

I hope you like them! (If not, just compare them to the horrible old icons).
