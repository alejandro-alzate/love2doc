Hey guys!

I'm quite new to LOVE / Lua but not that new to programming... and totally enthusiastic about games.
So I started a little mini game project to test out LOVE - actually because I was looking for a multiplatform engine which can be used for both free and commercial projects.

It will be a very simple game called "sukippu" - google translator says that this is the japanese writing for "rope skipping", if it's true... who knows
The name also says what it'll be about in the end - you have to skip ropes with an ever-increasing speed.

So, here's the github project: https://github.com/humansarepuppies/sukippu

Any comments, suggestions for improvements are very welcome

have fun and thanks for that lovely engine!
